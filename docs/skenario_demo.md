# Skenario Demo Singkat

1. Jalankan program dengan perintah `python main.py`.
2. Pilih menu 1 untuk menampilkan data obat awal.
3. Pilih menu 2 untuk menambahkan obat baru.
4. Pilih menu 3 untuk mencari obat berdasarkan kata kunci.
5. Pilih menu 4 untuk memperbarui stok obat.
6. Pilih menu 8 untuk membuat transaksi penjualan.
7. Masukkan ID pelanggan `PLG001` dan ID apoteker `APT001`.
8. Masukkan kode obat, jumlah, lalu ketik `selesai`.
9. Tampilkan struk transaksi.
10. Pilih menu 9 dan 10 untuk menampilkan laporan stok serta laporan penjualan.
